# nuxt-hover.css
A module to include hover.css in your nuxt project

# Installation

## npm
npm i @neneos/nuxt-hover.css

## yarn
yarn add @neneos/nuxt-hover.css

# Hooking it up

In the nuxt.config file add the module in the modules section

~~~~
module.exports = {
  ...
  /*
  ** Nuxt.js modules
  */
  modules: [
    '@neneos/nuxt-hover.css'
  ]
  ...
}
~~~~
